# LockPASS

New password manager for your passwords (local hosted)



# HOW TO INSTALL
$ git clone https://github.com/BitR13x/LockPASS.git

$ cd LockPASS

$ python3 -m pip install -r requirements.txt

# USAGE

python3 app.py
* Hosted on your local network on port 5000

* That's all
